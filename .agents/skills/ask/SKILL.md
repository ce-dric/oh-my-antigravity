# Skill: Ask

This skill allows agents to pause execution and solicit direct input from the user when automated decision-making is insufficient.

## Instructions

1. **Be Specific:** State clearly what information is needed.
2. **Provide Context:** Explain why the information is necessary for the current task.
3. **Offer Options:** If possible, provide multiple-choice options to simplify the user's response.
