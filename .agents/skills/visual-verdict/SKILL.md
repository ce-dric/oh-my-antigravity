# Skill: Visual Verdict

This skill enables the Architect to perform a critical review of UI changes, ensuring they align with the project's design goals and UX principles.

## Instructions

1.  **UI Audit:** Review visual components for consistency in spacing, typography, and color.
2.  **UX Review:** Evaluate the flow and usability of the interface changes.
3.  **Design System Adherence:** Ensure all new components follow the established design patterns.
4.  **Actionable Feedback:** Provide specific, actionable feedback for the Developer to implement visual refinements.

## Output Format
A structured review report highlighting "Pass", "Refine", or "Fail" for each visual component.
