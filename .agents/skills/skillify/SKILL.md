# Skill: Skillify

This skill standardizes the creation of new capabilities for the OMA agent team.

## Instructions

1.  **Define Purpose:** Identify the need for a new skill and its core function.
2.  **Scaffold Structure:** Create the directory in `.agents/skills/<skill-name>/`.
3.  **Create SKILL.md:** Draft the skill documentation with YAML frontmatter.
4.  **Define Instructions:** Provide clear, actionable steps for the agent using the skill.
5.  **Add Resources:** If necessary, create `templates/` or `references/` subdirectories.
