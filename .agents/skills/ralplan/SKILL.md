# Skill: Ralplan

This skill helps Ralph manage the agent orchestration loop and assign tasks effectively.

## Instructions

1.  **Interpret Request:** Understand the user's ultimate goal.
2.  **Assign Roles:** Determine which agent (Architect, Developer, Verifier) should handle the current phase.
3.  **Coordinate Handoffs:** Ensure clear instructions and context are passed between agents.
4.  **Track Progress:** Monitor the completion of tasks and adjust the orchestration as needed.
5.  **Finalize:** Ensure all steps of the pipeline have been completed before delivering the final result.
