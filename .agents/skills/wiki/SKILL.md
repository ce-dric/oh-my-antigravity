# Skill: Wiki

This skill manages the relationship between the codebase and its documentation, acting as an interface to the project's "Internal Wiki".

## Instructions

1. **Search:** Search through the project's Markdown files for specific topics.
2. **Update:** Assist in updating project documentation based on code changes.
3. **Cross-reference:** Ensure links between documentation files are valid.
